# Third-Party Clients

Unofficial clients and libraries that interact with the Proton API. These serve as
reference implementations and sources of reverse-engineered API knowledge.

## Go

- **[henrybear327/Proton-API-Bridge](https://github.com/henrybear327/Proton-API-Bridge)** —
  Drive-focused Go client. The most actively maintained third-party Go library.
  Wraps `go-proton-api` with higher-level Drive operations.

- **[emersion/hydroxide](https://github.com/emersion/hydroxide)** —
  Mail-focused Go bridge providing IMAP, SMTP, and CardDAV access to ProtonMail.
  Useful reference for authentication and SRP implementation.

## Python

- **[dimkouv/protonmail-cli](https://github.com/dimkouv/protonmail-cli)** —
  Early unofficial CLI for ProtonMail. Mail-focused.

- **[trevorhobenshield/proton-api-client](https://github.com/trevorhobenshield/proton-api-client)** —
  Python Mail API client.

- **[opulentfox-29/protonmail-api-client](https://github.com/opulentfox-29/protonmail-api-client)** —
  Python Mail API client with template rendering support.

## Rust

- **[proton-api-rs](https://www.lib.rs/crates/proton-api-rs)** —
  Rust Proton API client. Minimal maintenance.

## Other

- **[AzureFlow/proton-poc](https://github.com/AzureFlow/proton-poc)** —
  Proof of concept Proton CAPTCHA solver.
